# API Test Automation

This project demonstrates API test automation using **RestAssured**, **Java**, and **TestNG**.

## Technologies
- Java
- RestAssured
- TestNG
- Maven

## Test Scenarios
- GET /users API validation
- User count validation
- POST /users API test

## Project Structure

src/test/java
- base
- models
- tests

## How to Run Tests

Run tests using Maven:

mvn test
